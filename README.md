# Test Paper Scanner
OCR test paper recognition and pdf conversion API based on Flask and pytesseract

## System Requirements
-    Python 3.11

## Features
-    Flask
-    Pytesseract
-    PDF2Image
-    PDFMiner

## Installation
```bash=
    bash test_paper_scanner_install.sh
```
### Check Service
```bash=
    sudo systemctl status test_paper_scanner.service
```
